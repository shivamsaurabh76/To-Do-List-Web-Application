# To-Do-List-Web-Application

This is a simple and responsive To-Do List Web Application created using HTML, Bootstrap, and JavaScript. The application allows users to add, delete, and mark tasks as completed.

## Features

- Add new tasks with an input field and button.
- Mark tasks as completed.
- Delete tasks from the list.
- Edit existing tasks.
- Persist tasks using browser local storage.
- Filter tasks based on their status (all, active, completed).

## Technologies Used

- HTML5
- Bootstrap 5.3.3
- JavaScript
